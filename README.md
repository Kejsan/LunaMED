# LunaMed

## Project Info

LunaMed is a celestial science period tracker app, combining beautiful design with science-backed insights for comprehensive cycle and fertility tracking.

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

1. Clone the repository:

   ```sh
   git clone <YOUR_GIT_URL>
   cd LunaMED
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

## Technologies

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **State Management**: React Query, Context API
- **Backend (Auth/DB)**: Supabase

## Features

- **Cycle Tracking**: Log and predict menstrual cycles.
- **Insights**: Get personalized wellness recommendations.
- **Security**: Secure authentication and data privacy.
- **PWA Support**: Installable as a progressive web app.
